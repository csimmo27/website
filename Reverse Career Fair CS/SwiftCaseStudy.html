Case Study for Swiftris:

This was for my Computer Programming class in high school.

The goal of this program was to create a fully functioning Tetris-like game using only the provided materials. We were able to use all the preset images and sounds for the game, and then code the functionality. The program took a few weeks to complete with the many scenes, views, and objects that had to be created. It has a scoring and timing system, along with key and mouse movements for the blocks. Very fun project that took some time and critical thinking.

One of the problems I had while completing this program was not being able to stack the blocks on top of each other, they all kept going to the bottom of the screen and were stuck. I learned how to make

objects interact with one another much better when solving this problem.

Completed in Spring 2017
