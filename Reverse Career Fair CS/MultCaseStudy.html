Case Study for Multiplication Practice:

A program that acted as online flashcards, there is many ways to complete something like this, and there were many functions that we had to complete in this program.

The focus was generating random numbers between 0 and 12 and having the user enter their answer. There was a lot of extra things to do in this project such as; creating an option to set the number 12 to 2, for more simple problems, a timer so you only have 10 seconds to answer before it is counted as incorrect, and a scoring system to keep track of how many problems you answered correctly in a row. Once the user enters their answer, it is counted as incorrect or correct, and a newly generated problem is created.

One of the problems I had when creating this project was with the timing function. Once the timer reached 0, instead of counting the problem wrong and moving on to the next one, it would stop functioning completely. This was just a small issue with nesting loops incorrectly, this program helped me understand the use of multiple functions and button functionality within Swift.

Completed in Winter 2016
