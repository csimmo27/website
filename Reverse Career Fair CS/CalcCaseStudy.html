Case Study for Swift Calculator:

This was my very first project in the process of learning Swift, the iOS coding language. A simple program that took 3 days to complete in class.

The calculator was supposed to be fully functioning including simple operators. Numbers 0-9 are included, along with (+) (-) (x) (/) operators. The program stores information allowing you to perform multiple operations in a row, and there is a clear function that removes all the stored information.

One of the problems I came across while creating this was storing the information and performing multiple operations in succession. My calculator was stuck only performing one operation and not working with the new product. This was a simple fix and I learned a lot of Swift syntax and methods in the process of coding this.

Completed in Winter 2016
